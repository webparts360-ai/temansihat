Employee Licensing Requirements

When internal employees interact with a custom agent connecting to organizational data, Microsoft requires per-user licensing. The specific license depends on *where* the agent is surfaced.

  • ✔️
    Surfaced in M365 (Word, Teams, Copilot app) Requires a Copilot for Microsoft 365 license for every user interacting with the agent.
  • ✔️
    Surfaced as a Standalone Web App / Power Page Users need standard M365/Office 365 licenses, but the *tenant* requires Copilot Studio Message Capacity.
  • ✔️
    Development (The Builders) Developers need a Copilot Studio license to build and manage the connections.

Internal Cost Structure

Copilot for M365 ~$30 / user / mo
Copilot Studio (Tenant Base) ~$200 / mo
* Note: Prices are estimates based on standard commercial pricing and may vary by Enterprise Agreement.
🤝

External User Licensing (Customers, Suppliers, Partners)

External users do not require individual user licenses (like Copilot for M365). Instead, Microsoft uses a consumption-based model based on "Billed Messages".

Key Distinction: External users cannot access Copilot inside internal M365 apps. They must access the agent via a custom portal, website, or Teams Guest Access.

  • ✔️
    Copilot Studio Message Capacity You purchase "packs" of messages. A message is triggered when the external user sends a prompt and the agent responds.
  • ✔️
    Authentication (Optional) If connecting to secure supplier/partner data, you may need Power Pages authenticated user capacity, billed per login.

External Cost Structure

Tenant Base License 25,000 msgs incl.
Add-on Capacity Pack ~$200 / 25k msgs

Example: A customer support agent serving 10,000 customers who send an average of 5 messages a month = 50,000 messages.

Required: 1 Base ($200) + 1 Add-on ($200) = $400/month total.
